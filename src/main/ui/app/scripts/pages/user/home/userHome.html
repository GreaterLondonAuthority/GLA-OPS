Welcome to the Greater London Authority Open Project System (GLA-OPS)

GLA-OPS is a new, user-friendly online system used by the GLA's Housing and Land and Regeneration external partners to apply for grant funding and to manage land projects internally.

This new system currently enables you to:

  • Input and manage affordable housing grant allocations
  • Monitor land developments
  • Manage project assessment and approvals
  • Access simple reporting features depending on your role and permissions
  • Claim milestones and payments
You are logged in to GLA-OPS but you currently are not associated with any organisations.
Go to Manage Organisations to request access to an organisation.
If you have already made a request you will receive an email confirming your application in due course.
